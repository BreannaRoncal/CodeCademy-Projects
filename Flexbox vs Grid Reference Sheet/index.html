Flexbox vs Grid Reference Sheet

When to use which one

Flexbox is more one-dimensional, item layout is either horizontal or vertical. The widths/heights of the flex items are determined by their content, it grows or shrinks depending on the content and the available space. So, it's good for responsive websites.

Grid is two-dimensional with a main axis and a cross axis where you can place items into cells defined by rows and columns. The placement of items is precisely controlled, but the content does not grow or shrink.

Flexbox

Property Description
display: flex; Defines the flex container (inline)
flex-direction: direction; -row: left to right
-col: top to bottom
flex-wrap: direction; -nowrap (default): all items on one line
-wrap: all items will wrap onto multiple lines, from top to bottom
justify-content: alignment; Defines alignment along the main axis
-flex-start (default): items packed towards the start of the flex-direction
-flex-end: items are packed towards the end of the flex-direction
-center: items are centered along the line
-space-between: items are evenly spaced on the line, from start to end
-space-evenly: items are spaced equally between each other and from the edges.
align-items: alignment; Defines alignment along the cross-axis
-stretch (default): stretch to fill the container
-flex-start/end: see above description
-center: items are centered in the cross-axis

Grid

Property Description
display: grid; Defines a block-level grid
grid-template-columns/rows: values; Defines the columns and rows of the grid. The values are the size of each column/row and are space separated.
The values can be in px, named, repeat(num of times, values to repeat), and fr (fraction of free space of the grid container)
grid-template-areas: name; Defines a grid by referencing the names of the grid areas. Items are placed based on their grid-area properties.
-Repeating the name of the grid area expands the size of the area
-"." is an empty cell
-"none" means no grid areas are defined
gap: grid-row-gap grid-column-gap; Length values. Note: row-gap will default to column-gap, if none specified
justify-items: values;
align-items: values;		 justify-items lines grid items along the row axis, while align-items lines grid items along the column axis
-start: aligns items flush to the start edge of their cell
-end: aligns items flush to the end edge of their cell
-center: aligns items in the center of their cell
-stretch (default): fills the whole width of the cell
justify-content: values;
align-content: values;		 Use when the total size of your grid is less than the size of its grid container, i.e. when non flexible units are used. Then you can set the alignment of the grid within the grid container.
justify-content lines grid items along the row axis, while align-items lines grid items along the column axis
-start: aligns the grid flush to the start edge of the grid container
-end: aligns the grid flush to the end edge of the grid container
-center: aligns the grid in the center of the grid container
-stretch: resizes the grid items so the grid fills the wull width of the grid container
-space-around: places an even amount of space between each grid item, with half-sized spaces on the start and end edges
-space-between:places and even amount of space between each grid item, but no spaces on the start and end edges
-space-evenly: spaces in between and at edges are even
grid-column/row: start-line / end-line; Determines a grid item's location be referencing specific grid lines. -line: a number referring to a numbered grid line or a name to refer to a named grid line
-span (number or name): the item will span across the provided number or grid tracks or until it hits the next line with the provided name
-auto: auto placement, automatic span or a default span of one
-Default item will span 1 track if no value is declared
grid-area: area-name; Gives an item a name so it can be referenced by grid-template-areas property
-or used as a shorthand for grid-row-start + grid-column-start + grid-row-end + grid-column-end
place-self: align-self-value justif-self-value; Shorthand for for align-self and justify-self
-start/end/center/stretch. If only one value set, the second value will default to it
